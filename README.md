# creydrs AI Tools

Marketplace for my plugins

## Installation

### From the Claude Code Plugin Marketplace

1. **Add the marketplace:**
   ```bash
   /plugin marketplace add creydr/ai-tools
   ```

2. **Install a plugin:**
   ```bash
   /plugin install release-next-fix@creydr-tools
   ```

3. **Use the commands:**
   ```bash
   /release-next-fix:discover-problems
   ```
